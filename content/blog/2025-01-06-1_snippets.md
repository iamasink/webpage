basic typescript starter thing
```powershell
npm init -y; `
npm install typescript ts-node @types/node --save-dev; `
npx tsc --init; `
mkdir src; `
"console.log('hello world!');" | Out-File -Encoding utf8 src\index.ts; `
npx ts-node src/index.ts
```
